Technical overview

From responsibility
to working system.

Basel designed Shifa's central communication system using MQTT, allowing the website, Firebase database, Raspberry Pi interface, ESP32 controller, and actuators to exchange information through one coordinated architecture.

He organized the Node-RED implementation into three focused flows: a custom touchscreen user-interface flow, an alert flow for dose schedules and cup states, and a backbone communication flow for receiving, routing, saving, and redistributing system data.

This structure gave Shifa a clear communication center, allowing hardware and software modules to operate as one connected medication-management system rather than as separate components.
