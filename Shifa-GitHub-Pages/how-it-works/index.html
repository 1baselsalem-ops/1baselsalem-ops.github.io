02 · How it works

From a digital schedule to a physical dose.

Shifa connects its interface, cloud services, controller, and dispensing mechanism in one workflow.

01

Create the schedule

An admin assigns the medicine, compartment, dose, and time using the website or touchscreen.

02

Store and send

Firebase and Node-RED manage the schedule and send instructions through MQTT.

03

Dispense the dose

The ESP32 activates the correct stepper motor at the scheduled time.

04

Record the event

The system updates medication logs and can generate alerts for missed doses or system events.
